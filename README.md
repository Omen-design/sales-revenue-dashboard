# 📊 Sales & Revenue Analysis Dashboard

Interactive Power BI dashboard to analyze sales performance, revenue trends, and customer insights.

---

## 📷 Dashboard Preview

![Dashboard](screenshots/dashboard.png)

---

## 🔧 Tools & Technologies Used

* Power BI
* Excel
* SQL (for data extraction & transformation)

---

## 📊 Key Insights

* Delhi NCR generates the highest revenue contribution
* Sales quantity is highest in Delhi NCR market
* Revenue shows a declining trend in June
* Top 5 customers contribute a major share of total revenue
* Certain products dominate revenue generation

---

## 📁 Project Files

* `Sales & Revenue Analysis Dashboard.pbix` → Power BI dashboard file
* `screenshots/dashboard.png` → Dashboard preview image

---

## 🎯 Objective

The goal of this project is to:

* Analyze sales and revenue performance
* Identify top markets, customers, and products
* Track monthly revenue trends
* Support data-driven decision making

---

## 📌 Features

* Interactive filters (Year, Month)
* KPI cards (Revenue, Sales Quantity)
* Market-wise revenue & sales analysis
* Revenue trend over time
* Top customers & products analysis

---

## 🚀 How to Use

1. Download the `.pbix` file
2. Open it using Power BI Desktop
3. Explore dashboard using filters and visuals
